untracked -> Add(track & put file in staging area) -> Modified -> Add(put file in staging area) -> Commit GIT commands:-- 1. Intialize Git Repository -> git init 2. Track all files -> git add . 3. To Track specific file -> git add 'FILENAME' 4. To commit the changes -> git commit -m "Discription of this commit" - Always perform "git add ./'FILENAME'" before commit (Put file in staging area(A)) 5. To see list of commits - git log (to exit press q) 6. To add author details in commit -> a. git config --global user.name NamanPalliwal b. git config --global user.email palliwalnaman@gmail.com 7. To know status of files -> git status 8. To push GIT repo -> a. Create a repo on GITHUB b. git branch -M main c. git remote add origin 'link to that github repo.git (without '')' d. git push -u origin main
